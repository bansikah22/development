## How to Run the Application

1. **Clone the repository:**
    ```sh
    git clone https://github.com/bansikah22/development/learning/golang/web-server
    cd web-server
    ```

2. **Install dependencies:**
    Ensure you have Go installed on your machine. Then, run:
    ```sh
    go mod tidy
    ```

3. **Run the application:**
    ```sh
    go run main.go
    ```

4. **Access the application:**
    Open your web browser and navigate to `http://localhost:8443`.

5. **Build the application (optional):**
    If you want to build the application, run:
    ```sh
    go build -o web-server
    ```

6. **Run the built application:**
    ```sh
    ./web-server
    ```

Now your web server should be up and running!
